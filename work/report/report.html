Question 3: Give a brief explanation of how the browser, the HTTP protocol and the web server interacts.

Answer:Web browsers and servers communicate using TCP/IP. Hypertext Transfer Protocol is the standard application protocol on top of TCP/IP supporting web browser requests and server responses. Web browsers also rely on DNS to work with URLs.
